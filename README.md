# Webpack项目工程模版

## 功能

- 编译jsx、es6、scss资源
- 自动创建html页面
- 自动引入静态资源到相应html页面
- 开发模式实时编译和刷新浏览器
- 打包合并js、css
- 压缩js、css、html、图片
- 图片路径处理、压缩、CssSprite
- 对文件使用hash命名,做强缓存
- 支持Compass mixins 编写 scss
- js代码风格检查
- 自动化单元测试

## 命令

安装

    npm install

开发

    npm run dev

发布

    npm run build

测试

    npm run test
